## 任务详情

### 1. 信息源每日整理

**任务ID**: `c21c6466`
**执行时间**: 每天 18:00 (6:00 PM)
**类型**: Claude Cron (会话级)
**生命周期**: 7天自动过期（会话结束后仍运行）

#### 调用脚本
```
~/.claude/scripts/daily-info-organize.py
```

#### 功能说明

1. **清理重复文章**
   - 按 `source_url` 判断重复
   - 优先级: `score` 高 → `date` 新
   - 直接删除重复文件

2. **统计分类分布**
   - 分析一级分类（访谈/推特/文章/技术/投资/播客）
   - 检测二级分类（AI进展/Crypto/股票投资等）
   - 生成整理报告

#### 预期结果
```
📊 信息源每日整理报告
============================================================
🗑️  重复文章清理: X 篇
📁 当前分类分布: 各分类统计
📋 待优化二级分类: X 篇
============================================================
```

#### 日志位置
```bash
# 如配置系统级 cron，日志输出到
/tmp/info-organize.log
```
