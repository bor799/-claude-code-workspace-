# agent-reach

> **地位**: 互联网的感官系统，13+ 平台的搜索、读取、交互路由器，所有需要外部数据的技能都依赖它
> **逻辑**: 接收 URL/搜索词 → 平台识别（Twitter/微信/GitHub/通用） → 选择最优工具 → 输出结构化内容
> **约束**: 优先专用工具（Twitter 用 xreach，微信用本地工具），Jina Reader 兜底，需要网络代理才能访问部分平台

## 工具决策表

### URL 处理

| 平台/类型 | 首选命令 | 说明 |
|-----------|----------|------|
| 任意网页 | `curl "https://r.jina.ai/<url>"` | Jina Reader |
| Twitter/X | `xreach tweet <url> --json` | xreach CLI |
| 微信文章 | `~/.agent-reach/tools/wechat-article-for-ai/main.py` | 本地工具 |
| GitHub | `gh api /repos/owner/repo` | GitHub CLI |
| YouTube/B站 | `yt-dlp --dump-json "<url>"` | yt-dlp |

### 搜索

| 平台 | 首选命令 |
|------|----------|
| 通用搜索 | `mcp__exa__web_search_exa` |
| Twitter | `xreach search "<query>" --json` |
| GitHub | `gh search repos "<query>"` |

## 输入

| 类型 | 示例 |
|------|------|
| URL | `https://x.com/xxx/status/xxx` |
| 搜索词 | `Claude Code 技巧` |
| 平台 + 查询 | `Twitter AI news` |

## 输出

| 类型 | 格式 |
|------|------|
| 网页内容 | Markdown (Jina) |
| 推文 | JSON (xreach) |
| GitHub 数据 | JSON |

## 核心命令

```bash
# Twitter
xreach tweet "URL" --json              # 读取推文
xreach search "关键词" -n 10 --json    # 搜索

# 任意网页
curl "https://r.jina.ai/URL"

# GitHub
gh search repos "query" --limit 10

# 微信文章
cd ~/.agent-reach/tools/wechat-article-for-ai && python3 main.py "URL"
```

## 故障排查

```bash
# 检查可用通道
agent-reach doctor

# 配置代理
agent-reach configure proxy URL
```
