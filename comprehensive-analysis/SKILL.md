# 综合投资分析系统 v3.1

## 核心目标

> **运营本系统的核心目标**：判断当前这家公司/资产最关键的价值点，以及这个时间点是否该买入，以及买入的几个市场信号是什么。

### 分析框架：长期 + 短期

**长期视角**（资产价值）：
- ✅ 企业文化：创始人理念、组织架构、核心价值观
- ✅ 商业模式：护城河、竞争优势、可持续性

**短期视角**（市场时机）：
- ✅ 合适价格：PE/PB区间、安全边际、DCF估值
- ✅ 市场情绪：VIX、投资者情绪、资金流向
- ✅ 流动性环境：美联储政策、利率水平

## 5个核心分析 Agent

### Agent 1：财报深度分析 (tech-earnings-agent)

**文件**: `agents/tech-earnings.md`

**分析内容**：
- 16大模块：收入、利润率、现金流、指引、竞争格局等
- Key Forces识别
- Kill Conditions识别
- 估值矩阵（PE、PEG、EV/EBITDA、DCF）

**权重**: 30%（均衡分析时）

**触发场景**：
- "分析 NVDA 最新财报"
- "深度拆解 META 季报"
- "帮我看 Apple 这季度表现如何"

---

### Agent 2：价值投资评估 (value-investing-agent)

**文件**: `agents/value-investing.md`

**分析内容**：

- 自由现金流质量
- 护城河评估

## 交互式分析流程

### 第一步：启动分析

```
用户: /analyze [公司名] [代码]      # 股票分析
     /btc                            # BTC底部分析
     /fa 名创优品 9896.HK
     /买入分析 Lemonade LMND
```

### 第二步：确认与侧重点

```
系统: 📊 准备分析：[公司] ([代码])

请选择分析侧重点（影响4模块权重分配）：
A. 财务深度挖掘（财报50% + 其他各15%）
B. 价值投资评估（价值50% + 其他各15%）
C. 市场时机判断（情绪30% + 流动30% + 财报20% + 价值20%）
D. 全面均衡分析（各模块25%）

[BTC分析自动使用 crypto-bottom-agent]
```

### 第三步：执行分析

根据选择的侧重点，并行调用相应的 Agent，获取数据并分析。

### 第四步：综合决策

**输出3个核心问题的答案**：

1. **最关键的价值点是什么？**
   - 长期：企业文化/商业模式
   - 短期：催化剂/拐点

2. **现在该买吗？**
   - 买入 / 观望 / 卖出
   - 信心水平（高/中/低）

3. **买入信号有哪些？**
   - 估值信号：PE/PB回落到X倍
   - 基本面信号：XX指标改善
   - 市场信号：VIX>XX时
   - 流动性信号：美联储XX操作
   - 加密货币：MVRV-Z < 2, aSOPR < 0.98
