# 信息源清理技能

自动清理 Obsidian 信息源目录中的重复、低质量文件。

## 功能

- 🔍 自动检测重复文件（通过 source_url）
- 🗑️ 识别无标题/测试文件
- 📦 备份优先策略（删除前自动备份）
- 📊 生成清理报告

## 使用方法

### 通过 Claude Code

```
/cleanup-info-sources
```

或直接说：

```
清理信息源
```

```
删除重复文件
```

### 直接运行脚本

```bash
# 清理默认目录 (./信息源)
~/.claude/skills/cleanup-info-sources/cleanup.sh

# 清理指定目录
~/.claude/skills/cleanup-info-sources/cleanup.sh /path/to/info-sources

# 预览模式（不实际删除）
~/.claude/skills/cleanup-info-sources/cleanup.sh /path/to/info-sources dry-run
```

## 清理规则

| 优先级 | 类型 | 说明 |
|--------|------|------|
| P0 | 无标题文件 | 文件名包含"无标题"或"[no-title]" |
| P1 | 重复文件 | 相同 source_url 保留规范命名版本 |
| P2 | 时间戳命名 | 纯数字时间戳开头（如 091218_*） |
| P3 | 缺失元数据 | 缺少 source_url、title 等必要字段 |

## 安全保障

1. **备份优先** - 所有删除操作先移动到备份目录
2. **可追溯** - 生成详细报告记录所有操作
3. **保守策略** - 不确定时保留而非删除
