# Invest-Research Skills - 投资研究技能集

> 来源：[Day1Global/Day1Global-Skills](https://github.com/star23/Day1Global-Skills)

## 概述

这是一套专业的投资分析技能，覆盖科技股财报、宏观流动性、市场情绪和价值投资四大领域。

## 技能列表

### 1. tech-earnings-deepdive - 科技股财报深度分析

**功能**：机构级科技公司财报深度分析

**分析模块**（16个）：
- A. 收入质量分析
- B. 盈利能力分析
- C. 现金流分析
- D. 前瞻指引
- E. 竞争格局
- F. 核心 KPI
- G. 产品与新业务
- H. 合作伙伴生态
- I. 高管团队
- J. 宏观政策影响
- K. 估值模型
- L. 筹码分布
- M. 长期监控变量
- N. 研发效率
- O. 会计质量
- P. ESG 筛查

**6大投资哲学视角**：
- 质量复利（巴菲特/芒格）
- 想象力成长（Baillie Gifford/ARK）
- 基本面多空（Tiger Cubs）
- 深度价值（Klarman/Marks）
- 催化剂驱动（Tepper/Ackman）
- 宏观战术（Druckenmiller）

**触发示例**：
- "分析 NVDA 最新财报"
- "META 值得买入吗？"
- "从多个投资大师视角看 TSLA"

---

### 2. macro-liquidity - 宏观流动性分析

**功能**：全球宏观流动性环境分析与预测

**分析维度**：
- 央行政策跟踪
- 流动性指标监控
- 资金流向分析
- 风险偏好评估

**触发示例**：
- "当前宏观流动性如何？"
- "美联储政策对市场有什么影响？"
- "分析全球流动性状况"

---

### 3. us-market-sentiment - 美股市场情绪

**功能**：美股市场情绪与投资者行为分析

**分析内容**：
- 恐慌贪婪指数
- VIX 波动率
- 看涨/看跌比率
- 资金流向数据
- 社交媒体情绪

**触发示例**：
- "现在市场情绪怎么样？"
- "投资者是恐慌还是贪婪？"
- "分析当前市场情绪"

---

### 4. us-value-investing - 美股价值投资

**功能**：基于价值投资理念的股票评估

**四维价值评分**：
- 财务健康度
- 估值吸引力
- 盈利质量
- 成长潜力

**触发示例**：
- "从价值投资角度分析 AAPL"
- "这只股票被低估了吗？"
- "评估 MSFT 的投资价值"

---

## 协同使用

这些技能可以互相配合使用：

| 组合 | 场景 |
|------|------|
| `tech-earnings-deepdive` + `us-value-investing` | 财报分析后进行价值评分验证 |
| `tech-earnings-deepdive` + `macro-liquidity` | 考虑宏观流动性的财报分析 |
| `us-market-sentiment` + `macro-liquidity` | 结合流动性的市场情绪分析 |
