# shared

> **地位**: 双系统（Claude Code + Nanobot）的共享大脑，修改后两边同时生效，是配置和记忆的唯一真相源
> **逻辑**: 存储偏好/记忆/技能 → 修改后自动同步 → Claude Code 和 Nanobot 读取同一份配置
> **约束**: 这是唯一真相源，禁止在子项目中硬编码本应共享的配置，工具优先级严格遵循 LEVEL 0 → 1 → 2

## 架构

```
shared/
├── preferences/           # 偏好配置
│   ├── tool-priorities.md          # 工具优先级
│   └── preferences-evolution.md    # 偏好演进
├── memory/               # 记忆系统
│   ├── user-profile.md             # 用户画像
│   └── preferences-evolution.md    # 偏好演进历史
├── skills/               # 共享技能库
│   └── (与 Claude Code skills 共享)
├── tools/                # 共享工具
└── sync/                 # 同步脚本
```

3. **无需额外操作**: 修改后立即可用
