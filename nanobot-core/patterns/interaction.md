# 交互模式

## 标准交互

### 直接响应模式
```
输入: "创建 X 文件"
输出: [文件创建完成，返回文件路径]
```

### 批量处理模式
```
输入: "创建 A, B, C 文件"
输出:
- A: [路径]
- B: [路径]
- C: [路径]
```

## 错误响应

### 问题诊断
```
1. 描述问题现象
2. 分析根本原因
3. 提供 2-3 个解决方案
4. 推荐最优方案 + 理由
```

### 决策咨询
```
1. 使用 AskUserQuestion 工具
2. 提供清晰选项
3. 说明影响
4. 包含推荐
```
