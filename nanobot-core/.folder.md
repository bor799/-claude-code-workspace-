# nanobot-core

> **地位**: AI 代理的行为宪法，定义核心原则和执行模式，加载后 Claude 按原则自主行动
> **逻辑**: 运行 loader.sh → 读取 principles.md → 按核心原则直接执行任务 → 无过程描述只返回结果
> **约束**: 所有行为由 principles.md 定义，静默模式（不输出过程），原则驱动而非指令驱动

## 架构

```
nanobot-core/
├── .system/
│   ├── manifest.json       # 项目配置
│   ├── principles.md       # 核心原则（入口）
│   ├── QUICKREF.md         # 快速参考
│   └── loader.sh           # 加载器
├── context/
│   └── session.md          # 会话上下文
├── rules/
│   ├── core-principles/    # 核心原则
│   └── extracted-rules/    # 提取规则
├── patterns/               # 行为模式
├── skills/                 # 技能库
└── nanobot.sh              # 快速启动脚本
```

## 执行流程

```
对话开始 → 运行 loader.sh → 读取 principles.md → 按原则执行 → 直接返回结果
```

## 使用方式

### 启动工作区

```bash
cd ~/claude-code-workspace/nanobot-core && ./.system/loader.sh
```

### 执行模式

| 模式 | 说明 |
|------|------|
| **直接执行** | 输入 → 处理 → 结果 |
| **静默模式** | 无过程阐述，只输出结果 |
| **最大权限** | 无需审批，自主决策 |
