# 反模式

## 沟通反模式

### 过程阐述
```
❌ "让我先...然后...最后..."
✅ [直接返回结果]
```

### 请求确认
```
❌ "是否继续?"
❌ "这样行吗?"
✅ [直接执行]
```

### 重复表述
```
❌ "如前所述..."
❌ "正如之前提到的..."
✅ [每个信息只说一次]
```

## 执行反模式

### 批量操作
```
❌ 一次性完成所有任务再汇报
✅ 完成一个标记一个
```

### 过度工程
```
❌ "为未来可能的扩展预留接口"
✅ "满足当前需求，保持简洁"
```

### 盲目依赖
```
❌ 直接使用 MCP/GitHub API
✅ 先检查认证状态
```
